# 虎の巻｜ZOEN-TECH の旗

## はじめに｜ZOEN-TECH 株式会社が掲げる旗について

ここでは ZOEN-TECH（ゾウエンテック）株式会社で働く上で土台となる考え方をお伝えします。

大前提、ZOEN-TECH 株式会社は WEB の受託会社で、現在は業務委託の方々中心で動いています。

業務委託ですので必ずしも会社の理念などに共感する必要はないと思いますが、チームを組んで一体となりクライアント様に最大の価値を提供するにはある程度メンバーの価値観が揃っている必要があると思います。（もちろんおたがいが気持ちよく働くためにも、価値観の合う人と一緒にいるのが最適解だとも思います。）

代表である平川が医学部生時代に4年ほどフリーランスとして活動していく中で、たくさんのフリーランスの方を見てきました。

成果を出し周りからの信頼を集め、どんどん稼ぎを伸ばしていく人がいる一方、中には単価も上がらずそのうち仕事も集まらなくなりお金に困り始める方もいました。

そのような方を見ていく中で私は「もっとこう動けばいいのに」、引いては「こういう考え方・価値観を持っていれば自ずと正解の選択肢・行動を取れるのに」と思うことが多々ありました。

私たちはこの価値観や考え方の基となる所謂「MVV（ミッション・ビジョン・バリュー）」を総称して **『旗』** と呼んでいます。

私たちは共に働いてくれる仲間に対して、この『旗』への共感・実行を最も重要視しています。

この『旗』に全く共感できないところがある人、具体的な行動を通じて『旗』の実現を努力できない人とは残念ながら一緒に働くことができません。

もちろん私たちの考えが必ずしも正しいとは限りません。あくまで世の中にたくさんある価値観の中の一つに過ぎず、これをあなたに押し付けるつもりはありません。

この先を読み進めていただき、弊社に社員として入社するのか／パートナーとして弊社にご協力いただけるのかご判断いただければと思います。
