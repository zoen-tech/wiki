# 基本の使い方

Google Chat の基本操作と、ZOEN-TECH での使い分けの考え方をまとめます。

## 🏠 スペースとは

**スペース** は、特定のプロジェクトやチームで会話をするための「部屋」です。

### スペースの種類

| 種類 | 用途 | 例 |
|---|---|---|
| プロジェクト全体 | プロジェクトメンバー全員 | 《PJ》カブクリ_全体 |
| チーム別 | 機能・職能ごと | 《Team》エンジニア |
| 雑談 | 業務外の会話 | 《雑談》ランチ |
| DM | 個人間 | （個別） |

!!! note "DMはサマリ・通知の対象外"
    DMはプライバシー保護のため、サマリBotや通知Botの対象外です。重要な決定事項はスペースで行ってください。

## 💬 メッセージの基本

### 投稿
- メッセージ欄にテキストを入力 → Enter で送信
- 改行は **Shift + Enter**
- 文字装飾：**太字** (`*太字*`)、_斜体_ (`_斜体_`)、~取り消し線~ (`~取り消し~`)、`コード` (`` `コード` ``)

### 編集・削除
- 送信後のメッセージは右クリック → 「編集」または「削除」
- 編集すると「（編集済み）」と表示される

### 引用返信
- メッセージにマウスを乗せて「メッセージから返信」
- 元のメッセージが引用された状態で返信できる

## 😀 絵文字リアクション

メッセージへのリアクションは **絵文字** で行います。

### 使い分けの基本

| 絵文字 | 意味 |
|---|---|
| 👍 | 了解した・確認した・賛成 |
| 👀 | 見ました（あとで対応する） |
| ✅ | 完了した |
| ❤️ / 🎉 | ナイス・お祝い |
| 🤔 | 検討中・疑問あり |
| 🙏 | お願い・感謝 |

!!! tip "返信の代わりに使う"
    「了解しました」だけのメッセージは情報量がなく流れを増やすだけです。
    **返信不要なら絵文字リアクションで済ませる** のがZOEN-TECH流です。

## 📎 ファイル共有

- 添付アイコンからローカルファイルをアップロード可能
- Google Drive 連携で Drive 内のファイルを直接共有可能
- 大きなファイルや継続的に編集するファイルは Drive に置いてリンク共有を推奨

## 🔍 検索

- 上部の検索バーから過去のメッセージを検索可能
- スペース指定・送信者指定・期間指定で絞り込めます
- **日付フィルタ**: `before:2026-01-01`, `after:2026-01-01`
- **送信者フィルタ**: `from:t.hirakawa`

## 📱 通知設定

### 推奨設定

| シーン | 設定 |
|---|---|
| 業務時間中 | すべて通知ON |
| 業務時間外 | おやすみモード（DND）を活用 |
| 集中したいとき | 個別スペースの通知をミュート |

スペース名右クリック → 「通知設定」から個別調整できます。
