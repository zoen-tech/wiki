# スラッシュコマンド

ZOEN-TECH では Google Chat 内で **スラッシュコマンド** を使って各種Bot機能を呼び出せます。

メッセージ欄に `/` を入力すると、利用可能なコマンドのサジェストが出ます。

## 🤖 ゾウエンくん（お手伝いbot）

ZOEN-TECH 公式の社内Bot。各スペースに追加して使います。

### `/task` - 自分のタスク一覧を表示

**使い方：**
```
/task
```

実行したスペースに紐づく **自分宛のGoogleタスク** がカテゴリ別に表示されます。

**表示内容：**

| カテゴリ | 内容 |
|---|---|
| 🟠 **期限切れ**（要対応） | 期限を過ぎているタスク |
| 🟡 **期限未設定**（要期限設定） | 期限が設定されていないタスク |
| 🟢 **対応中**（登録済み・期限内） | 期限内のタスク |

各タスクには **通し番号**・**期限**・**チャットメッセージへのリンク** が表示されます。

!!! note "対象はそのスペース"
    `/task` は、実行したスペースに紐づくタスクのみ表示します。
    他スペースや個人タスクは含まれません。

#### 表示例

```
📋 平川拓実さんのタスク一覧
━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━

🟠 期限切れ（要対応）

【1】 LP のCVボタン色を緑→オレンジに変更
    📅 期限: 1/3（97日経過）
    🔗 https://chat.google.com/...

【2】 マンジャロ副作用の解説投稿案作成
    📅 期限: 2/20（49日経過）
    🔗 https://chat.google.com/...

━━━━━━━━━━━━━━━━━━
🟡 期限未設定（要期限設定）

【3】 Google口コミ投稿の運用ルール検討
    🔗 https://chat.google.com/...

━━━━━━━━━━━━━━━━━━

```

#### Phase 2 機能（実装予定）

完了したタスクをBotに伝える機能を実装予定です：

```
（タスク一覧のスレッドに返信）
1完了
3完了
```

→ Botが該当タスクをGoogle Tasksで完了マークします。

## 🚧 今後追加予定のコマンド

### `/summary` - 今日のチャットサマリを表示

実行したスペースの本日のサマリ（決定事項・未解決事項）を表示する予定。

### `/status` - プロジェクトの進捗状況を表示

スペースに紐づくプロジェクトの進捗（タスク完了率等）を表示する予定。

## ❓ よくある疑問

### Q. コマンドが出てこない・反応がない

A. **Botがそのスペースに追加されていない** 可能性があります。
平川に連絡して、Botをスペースに追加してもらってください。

### Q. 別のスペースのタスクも見たい

A. 現状は実行したスペースのみ対応。
複数スペースを横断するコマンド（`/task --all` 等）は将来追加予定です。

### Q. 自分以外のメンバーのタスクも見られる？

A. いいえ。`/task` は実行したユーザー本人のタスクのみ表示します。
他の人のタスクはプライバシー観点から表示しません。
