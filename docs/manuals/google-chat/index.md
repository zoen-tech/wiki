# Google Chat 運用ガイド

ZOEN-TECH では、社内コミュニケーションの中心ツールとして **Google Chat** を使っています。
チームでの情報共有・意思決定・タスク管理を円滑にするため、以下のルール・ガイドラインを設けています。

!!! tip "迷ったらこれだけ（4つの基本）"
    1. **新しい話題はスレッドで始める**（流れない）
    2. **メンションされたら12時間以内に何かしらの反応を返す**（返信 or リアクション）
    3. **5分以上かかる依頼は1日以内にGoogleタスクに登録＋期限設定**
    4. **判断・決定はチャットに残す**（口頭の決定もチャットに書き起こす）

---

## 🎯 このガイドの目的

- **情報の流出を防ぐ** — 決定事項やタスクが流れて見失われないようにする
- **返信漏れを減らす** — 誰がボールを持っているか明確にする
- **ナレッジを蓄積する** — 日々のやり取りから学びを資産化する
