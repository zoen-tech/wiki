# サマリBot（自動化）

ZOEN-TECH では Google Chat のやり取りを **AIで自動解析** し、決定事項・未解決タスク・共有事項を整理してリポジトリに蓄積する仕組みを運用しています。

## 📖 概要

### この仕組みで解決すること

- チャットで流れてしまう **決定事項の記録漏れ**
- 誰がボールを持っているか分からなくなる **タスクの見失い**
- メンションされたのに気づかない **返信漏れ**

---

## 🔄 動作の流れ

### サマリ生成（1日3回）

```
8:00 / 17:00 / 22:00
  ↓
① Google Chatからメッセージを取得
② AIが内容を分析し、サマリを生成
   ・決定事項（経緯付き）
   ・未解決・要確認
   ・共有事項
③ サマリをリポジトリに保存
   → docs/chat-summary/YYYY-MM-DD.md
④ 📊 サマリ更新通知をグループに送信
```

### リマインド通知（朝8:00のみ）

```
以下を検知してGoogle Chatに通知
  ・📩 返信漏れ（12時間以上未返信＋未リアクション）
  ・📋 タスク化リマインド（発生から24h経過＆未登録）
  ・📅 期限未設定リマインド
  ・⚠️ 期限切れリマインド
```

## 👤 メンバーがやること

### 普段の運用（特別なことは何もありません）

- **普通にGoogle Chatを使ってください。** 特別な書き方やフォーマットは不要です
- 決定事項やタスクはAIが自動で判別します

### 通知が来たときの対応

#### 📩 返信漏れ通知

自分宛のメンションに **12時間以上** 返信もリアクションもしていない場合に通知されます。

**対応方法：**
- 返信が必要 → 該当スレッドに返信する
- 返信不要 → 元メッセージに 👍 等のリアクションを付ける

#### 📋 タスク化リマインド

チャットで発生したタスクが **24時間** 経ってもGoogle Tasksに登録されていない場合に通知されます。

**対応方法：**
1. 通知内のリンクから該当メッセージを開く
2. メッセージの「︙」メニュー → 「タスクに追加」
3. **期限を必ず設定して保存**

詳細は [タスク管理ルール](task-management.md) を参照。

#### 📅 期限未設定リマインド

Googleタスクに登録されているが期限が未設定の場合に通知されます。

**対応方法：**
- Googleタスクを開いて期限を設定する
- そもそも対応不要なら、タスクを完了にする

#### ⚠️ 期限切れリマインド

Googleタスクの期限を過ぎている場合に通知されます。

**対応方法：**
- 完了している → タスクを完了にする
- まだ対応中 → 期限を再設定する

#### 📊 サマリ更新通知（毎回）

サマリが更新されたことの通知です。対応は不要ですが、決定事項や未解決事項を確認したい場合はリポジトリのサマリファイルを参照してください。

## 🔒 セキュリティ・プライバシー

- **患者の個人名はイニシャルに変換** されます（例：田中太郎様 → T.T様）
- サマリはチャットスペースに対応するリポジトリに格納されます
- リポジトリのアクセス権 = チャットスペースの参加者と一致するように管理されています
- **DMはサマリ対象外** です

## ❓ FAQ

### Q. 通知がうるさい場合は？

A. 平川に連絡してください。通知の頻度や対象を調整できます。

### Q. サマリの内容が間違っている場合は？

A. AIが生成しているため、誤りがある可能性があります。気づいたら平川に連絡してください。

### Q. 自分のメッセージをサマリ対象外にしたい場合は？

A. 現時点では個別除外の機能はありません。必要があれば平川に相談してください。

### Q. 新しいチャットスペースを作った場合は？

A. 自動で検知されます。平川が格納先リポジトリを設定するまではサマリ対象外です。

### Q. サマリはどこから見られる？

A. 各リポジトリの `docs/chat-summary/YYYY-MM-DD.md` に格納されています。
GitHub にアクセスして該当ファイルを参照してください。
